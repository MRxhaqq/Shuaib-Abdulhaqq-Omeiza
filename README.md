# Shuaib Abdulhaqq — Portfolio

Personal portfolio website built with React, Tailwind CSS, and Framer Motion.

## Live Demo

[View live](https://mrxhaqq.vercel.app)

## Features

- Animated welcome screen with progress bar
- Custom glowing dot cursor
- Light / dark mode toggle with smooth transition
- Animated hero section with terminal avatar and typing effect
- Scroll-triggered section animations
- About section with personal story and highlight cards
- Skills section with animated progress bars and tech stack
- Project showcase cards with hover effects
- Working contact form powered by EmailJS
- Scroll to top button
- Fully responsive — mobile, tablet, desktop
- SEO meta tags and Open Graph support

## Tech Stack

- React 18
- Tailwind CSS v4
- Framer Motion
- EmailJS
- Vite
- Lucide React

## Getting Started

### Installation

```bash
git clone https://github.com/MRxhaqq/Shuaib-Abdulhaqq-Omeiza.git
cd portfolio
npm install
```

### Environment Variables

Create a `.env` file in the root:

```
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

### Run locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
